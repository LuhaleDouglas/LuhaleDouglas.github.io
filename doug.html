PROGRAMMING LANGUAGE OVERVIEW
Program
  • Model of complex system
    • Model: simplified representation of salient features of something, either tangible or abstract
    • System: collection of components that work closely together
  • Sequence of instructions expressed in specific programming language
    • Syntax: grammatical rules for forming instruction
    • Semantics: meaning / interpretation of instructions

Programming

It is the process of writing these instructions in a language that the computer can respond to and that other programmers can understand.

Programming language

The set of instructions that can be used to construct a program

Why Programming Languages?

  • Need to issue instructions to computer’s hardware
  • Need to indicate tasks to be carried out
  • Programming languages allows you to express yourself more precisely than natural language
  • Need to indicate problems to be solved

High-level language paradigms

  • Procedural
  • Object-oriented
  • Functional
  • Logic
  • Others

Procedural paradigm

  • Consists of writing a list of instructions (or actions) for the computer to follow, and organizing these instructions into groups known as functions.
  • Programs are instructions, executed in specific order, to manipulate data
  • Expressed as commands
  • Focus is on action or task

Major procedural languages

    FORTRAN
  • (FORmula TRANslator)
    COBOL
  • (COmmon Business Oriented Language)
    BASIC
  • (Beginner’s All-purpose Symbolic Instruction Code)
  • C
  • Pascal
  • Ada

Object-oriented paradigm

  • Programs are simulations of real-world behavior
  • Expressed by identifying objects and asking them to perform certain tasks
  • Focus is on object, THEN on task
  • A computer language is object-oriented if they support the four specific object properties called abstraction, polymorphism, inheritance, and encapsulation.

Major “object-oriented” languages

  • Pure OO languages
    • Smalltalk
    • Java
  • Hybrid languages
    • C++
    • Visual Basic
