DATA STRUCTURE
Is a representation of data and the operations allowed on the data.
A data structure is a way to store and organize data in order to facilitate the access and modification.

Data structure gives method of representing the logical relationship between individual data elements related to solution of given problem.

BASIC TYPES OF DATA STRUCTURE
Anything that can store data, can be called DATA STRUCTUREeg integer, float, boolian, character. They are known as primitive data structure.

We also have complex data structure which are used to store large and connected data, they are known as abstruct data structure , they include linked lists, tree, graph, stack and queues.
